Instructiuni

Un genist (expert in bombe) se plimba pe strada, cand aude un strigat si un ticait de bomba din hotelul alaturat. Isi da seama ca persoana cealalta este foarte speriata si vrea sa o ajute sa dezamorseze bomba.

  1. Genistul nu poate intra in camera de hotel, dar va comunica cu civilul prin avioane pe care le trimite prin fereastra hotelului.
  2. DOAR genistul pune intrebari prin avioane, iar cealalta persoana raspunde la intrebarile primite. E prea speriata ca vina cu idei/intrebari.

Dezamorsarea bombelor pentru incepatori :D

Afla informatiile de la civil si ghideaza-l conform instructiunilor tale

AN FABRICATIE MODEL INSTRUCTIUNI
1945 SMARTBOMB
  1. Daca nu exista niciun fir alb, taie firul rosu.
  2. Daca exista un fir alb, taie firul verde
  3. Schimba simbolul din dreapta in leu
2022 ROBOMB
  1. Daca exista un fir alb, lipeste la loc firele galbene cu banda adeviza
  2. Schimba simbolul din dreapta in papagal.
  3. Taie firul negru
  4. Taie firul alb.

Ca sa obtii informatii de la civil:

Din ce an si ce model este bomba ?

Culoare avion: Verde

Method: GET

URL: https://exemplu-api-fmi.herokuapp.com/bomb

Parametrii: nimic

OBS: Citim o informatie - READ

Exista un fir alb ?

Culoare avion: Verde

Method: GET

URL: https://exemplu-api-fmi.herokuapp.com/check?year=2022&model=ROBOMB&wire=white

Parametrii in URL: ?year=2022&model=ROBOMB&wire=white

OBS: Citim o informatie - READ

Lipeste la loc firele galbene

Culoare avion: ALBASTRU

Method: POST

URL: https://exemplu-api-fmi.herokuapp.com/bomb/create

Parametrii in URL: nimic

Parametrii in BODY: (obiect JSON) { wire= "yellow", "how" = "tape" }

OBS: Creem o conexiune noua intre fire - CREATE

Schimba simbolul din dreapta in papagal

Culoare avion: PORTOCALIU

Method: PUT

URL: https://exemplu-api-fmi.herokuapp.com/bomb

Parametrii in URL: nimic

Parametrii in BODY: (obiect JSON) { "symbol":"lion" }

OBS: Modificam simoblul existent --> UPDATE

Taie firul negru

Culoare avion: ROSU

Method: DELETE

URL: https://exemplu-api-fmi.herokuapp.com/bomb

Parametrii in URL: nimic

Parametrii in BODY: (obiect JSON) { wire= "negru" }

OBS: Stergem(taiem) ceva existent --> DELETE

OBS: Am facut operatii de Create, Read, Update Delete. Acronimul este CURD
