Rezultatul din primul laborator : Laborator 1

Pe partea de HTML am invatat :

  1. Structura unei pagini HTML
  2. Tag Titlu-pagina : <title>Pagina mea</title>
  3. Incarcarea unui favicon
  4. Tag heading: h1,h2...h6 <h1>Cea mai noua descoperire in medicina !</h1>
  5. Tag paragraf : p <p>Eu sunt un paragraf. O sa ma folosesti foarte des</p>
  6. Tag span <p>Eu sunt anul 3 la <span style="color:red">Mate-Info</span></p>

    Eu sunt anul 3 la Mate-Info

  7. Tag pentru imagini : img <img src="calea catre imagine" alt="mesaj care apare cand nu avem internet sau src nu duce nicaieri" />
  8. Comentarea codului : <!-- Acesta este un comentariu; -->
  9. Liste neordonate : <ul></ul>
  10. Liste ordonate : <ol></ol>
  11. Link : <a></a>
  12. Input : <input></input>
  13. Input : <table></table>

Tema:

Sa lucrati la CV-ul vostru folosind tag-urile invatate in primul laborator. Cei care ati predat tema din laborator nu trebuie sa mai trimiteti nimic. Deadline: Urmatorul laborator, finalul zilei.

Pe mail nu puteti trimite direct fisiere html, css sau js.

Folositi WeTransfer sau Google drive(cu permisiune de vizualizare pe link)

O sa va trimit un mesaj de confirmare pentru tema trimisa in aceeasi zi; Daca nu primiti mail de confirmare inseamna ca n-am primit tema

Inainte sa trimiteti o tema sau un proiect, validati fisierul HTML folosind HTML validator. Se pierd puncte pentru erori de validare
