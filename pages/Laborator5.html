Javascript - Introducere

Pana acum am folosit HTML ca sa definim structura unei pagini si CSS pentru modul in care site-ul arata.

Javascript este un limbaj de programare pe care il folosim ca sa adaugam interactiune(sa facem site-ul dinamic).

Site web vs Aplicatie Web

Este important sa facem urmatoarea distinctie:

  • Site web - este static, de prezentare. (poti doar sa vezi un continut, care este de fiecare data acelasi). Ex: site cu recenzii despre carti
  • Aplicatie web - este complexa, interactiva. (ex: facebook, trello, emag etc)

Cum folosim JavaScript intr-o pagina web?

In fisierul nostru HTML, la final de body, "legam" script-ul, specificand sursa 
                    // la fel ca la CSS, proprietatea src este "calea" catre fisier
                          <script src="myscripts.js"></script>

Trebuie sa adaugam script-ul la final de body pentru ca vom interactiona cu elementele din pagina. Daca am adauga script-ul in head sau la inceput de body, este posibil ca elementele sa nu existe in pagina cand rulam script-ul

Observatii

  1. Utilizarea ; este optionala, dar recomandata pentru calitatea codului
  2. Este bine sa nu lasam variabile undefined. Daca nu vrem sa atribuim o valoare la initializare, atunci folosim null. Motivatie: Cand cineva vede o variabila undefined, nu stie daca am uitat sa o initializam sau chiar am vrut sa nu aiba o valoare momentan

Functii 

                        // definim functia
                        function adunare(primulNumar,alDoileaNumar){
                            // returnam suma celor 2 numere
                            return primulNumar + alDoileaNumar;
                        }

                        // apelam functia:

                       var suma = adunare(10,20);
                       console.log("suma este", suma); // 30

                       // Daca apelam functia cu mai multi parametrii decat am definit initial, 
                        // parametrul in plus este ignorat
                       var suma2 = adunare(10,20,30); // 30 este ignorat; sunt adunate doar primele 2 numere

                       // Daca apelam functia cu un parametru in minus in schimb, se intampla ceva interesant;
                       // alDoileaNumar o sa aiba valoarea default, adica undefined
                       // javascript incearca sa adune numarul 10 cu string-ul undefined
                       // ca sa adune cele 2 valori, numarul 10 este convertit la string-ul "10";
                       // "10" + "undefined" este concatenare de stringuri
                       // deci o sa avem "10undefined" ca string

                       var sumaUndefined = adunare(10,undefined);
                       console.log("sumaUndefined este ", sumaUndefined) // sumaUndefined este 10undefined

                       // ca sa evitam aceasta problema, avem 2 variante
                       // 1) Verificam daca s-a transmis o valoare in cadrul functiei:

                       function adunare(primulNumar,alDoileaNumar){
                        // returnam suma celor 2 numere
                        if(typeof primulNumar === "undefined")
                            primulNumar = 60 // o valoare default, pe care o vrem noi

                        if(typeof alDoileaNumar === "undefined")
                            primulNumar = 28 // o valoare default, pe care o vrem noi

                        return primulNumar + alDoileaNumar;
                    }

                    // 2) Incepand din versiunea ES6 de javascript, 
                    // exista un mod mai simplu pentru a da o valoare default

                    function adunare(primulNumar = 60 ,alDoileaNumar = 28){
                        // returnam suma celor 2 numere
                        return primulNumar + alDoileaNumar;
                    }

                    // valoarea default este folosita doar daca noi nu apelam functia cu parametrul respectiv

                adunare(15) // functia este apelata cu primulNumar = 15 si alDoileaNumar = 28(default)
                adunare() // functia este apelata cu primulNumar = 60(default) si alDoileaNumar = 28(default)
                adunare(45,50) // sunt folositi parametrii dati de noi, nu cei default

Instructiunea IF; == vs ===

                        // == verifica valoarea, fara sa tina cont de tipul de date
                        5 == 5  // true 
                        5 == "5" // true

                        // apar unele situatii neobisnuite

                        0 == false // true
                        0 == [] //true
                        0 == {} // false 

                        // === verifica valoarea, inclusiv tipul de date
                        5 === 5  // true 
                        5 === "5" // false pentru ca sunt tipuri de date diferite

                        // Instructiunea IF 

                        if( a >= b){
                            console.log("a este mai mare sau egal cu b");
                        }
                        else{
                            console.log("b este mai mare decat a");
                        }
