Proiect HTML + CSS

Proiectul se va preda cel tarziu in laboratorul 8: 6-10 aprilie

Puteti preda proiectul si mai devreme, dar trebuie sa respecte cerintele eliminatorii

Dupa data de 10 aprilie, proiectele trimise cu intarziere pot primi MAXIMUM 0.5puncte (50% din punctajul initial)

UPDATE: Pentru trimiterea proiectelor, va rog sa folositi github si sa-mi trimiteti link catre proiect

Cerinte proiect

Exista 2 tipuri de cerinte :

  • Eliminatorii - nu se prezinta proiectul fara ele; Sunt notate cu rosu
  • Recomandate - va ajuta sa acumulati puncte; Puteti prezenta si fara ele

Punctaj toatal proiect HTML si CSS -- 1 punct

Generale → Nu au punctaj, dar sunt obligatorii

  1. Fisier CSS extern (NU stil inline sau intern)
  2. Sa aveti minim 4 pagini HTML (ex: Contact, Despre Noi, Produse, Oferte speciale etc)

HTML

  1. Folosirea tag-urilor: h1,h2...h6,p,img,audio,a,span,ul,ol,li,tabel,iframe → 0.1p
  2. Incarcarea unui favicon → 0.015p
  3. Validarea HTML (fara erori si fara warning-uri) → 0.015p

CSS→ Depinde de cati selectori folositi; 0.25 - 0.6

  1. Meniu dropdown → 0.1p
  2. CSS VALID - → 0.015p CSS Validtor
  3. Utilizarea animatiilor din CSS → 0.05p pentru fiecare animatie distincta(chiar sa fie diferita, nu doar o alta culoare sau diferent f.f mici). Se poate obtine cel mult 0.1 pentru animatii in total
  4. Selectori diferiti de CSS ( dupa tag, clasa, id, copil al unui alt element, copil direct, al n-lea copil de tipul unui tag etc → 0.01p pentru fiecare selector diferit
  5. Proprietati CSS de folosit: (sunt exemple, puteti folosi si alte proprietati): width, height, color, background, font-size, border, padding, margin, display, overflow, z-index etc ; Maxim 0.1p in total → 0.005 pentru fiecare proprietate folosita

Responsive (sa arate bine pe diferite dimensiune de ecran, inclusiv tableta/telefon) → 0.35

  1. Media query pentru dimensiuni mari, medii si mici de ecran (adica sa aveti 3 @media, destul de diferite intre ele; ex: ascunde elemente la dimensiune de telefon, schimbati dimensiunea scrisului etc; NU doar o culoare de fundal) → 0.2p
  2. Folosire rem, em, procentaje, vh, vw - dupa necesitat → 0.005 pt fiecare unitate de masura

Altele

  1. structurarea proiectului in foldere (imaginile intr-un folder de imagini, stilurile intr-un folder de stiluri, videoclipurile intr-un folder de videoclipuri, scripturile intr-un folder de scripturi etc.). CSS-ul sa fie impartit in mai multe fisiere( CSS general, CSS pentru paginile speciale, CSS pentru diverse media cum ar fi cel pentru printare). Toate fisierele sa aiba nume sugestive (de exemplu stil_general.css, home.css, stil_general_media_700_1000.css) → 0.05p
  2. Formular de contact sau Login (utilizare input-uri cu type diferit; ex: input text, password, email, select, textarea, color etc) → 0.05p
  3. folosirea codurilor html pentru caractere speciale &cod; ex: pentru "mai mare sau egal", "sageata" / ce simboluri vreti → 0.02p
  4. folosirea unui font custom pentru scris → 0.03p Exemplu
  5. Utilizare CSS flex → 0.05p
  6. Utilizare CSS grid → 0.1p
  7. Introducerea de butoane catre facebook, twitter → 0.05p
  8. Adaugarea site-ului pe net (exita destule variante gratuite; cauti pe net sau ma puteti intreba oricand si va explic) → 0.05p
  9. Folosirea Boostrap intr-o pagina ( !! Atentie - este permis intr-o singura pagina, nu puteti face tot proiectul folosind Bootrstap) → 0.05p
